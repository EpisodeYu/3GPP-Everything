### 2.2 `eval/scripts/langfuse_cleanup_and_push.py`（新）

三段开关 flag：

- `--clean-traces`: 删全部 Cloud trace（M7.5 期间 stale）
- `--push-dataset`: upsert tgpp-golden-v1 175 items（用 push_golden_to_langfuse）

  + 通过 `_link_trace_to_dataset_run` 关联 dataset_run_item + push 10 个 score

`--dry-run` safety guard。

**坑点**：第一版 `_push_run` 误用 `make_eval_trace_id` (deterministic UUID seed
不创建 trace 实体) + `push_run_score` → 175 个 orphan score Cloud 全过滤了。
修正后用 runner 同一套 trace 创建管线。

### 2.3 `eval/settings.py` + `eval/negative_judge.py`（修）

tool_choice"），但 ragas 走 JSON schema OK。两个 judge 用途分开配置避免混淆。
