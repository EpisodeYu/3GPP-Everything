## 3. 自主决策记录

下列决策在 §5 触发线之内、未停下问人，按 `CLAUDE.md §4.3` 在此点名：

1. **bcrypt 直接调用替代 passlib** — `pyproject.toml` 仍列 `passlib[bcrypt]`，但实际代码不再用 passlib（passlib 1.7.4 在 Python 3.13 / bcrypt 5 下触发 `crypt` 模块 DeprecationWarning，且 passlib 已不维护）。bcrypt 直接调用 hash/verify 行为等价，且无外部依赖风险。
2. **ruff 全局 ignore `B008`** — FastAPI 项目 `param = Depends(...)` 默认值用法跟 `B008` 冲突；项目所有路由通用，整目录关而非每行 noqa。
3. **`scripts/dev/*.py` E501 per-file-ignore** — main 上 `scripts/dev/diff_chunks_meta_schema.py` 既有长中文行 lint fail（pre-existing，与 alembic 既有忽略一致原因），仿 alembic 加 per-file-ignore。
4. **refresh token DB hash 用 SHA-256 而非 bcrypt** — refresh 校验是 hot path 需要 O(1) 查表；token 本身已是高熵 JWT，SHA-256 hex 足够；不存敏感明文。
5. **`audit_logs.extra` 不写 token / content** — Q5 决策的反向断言已写进 `test_audit_does_not_contain_chat_content_in_m46`。
6. **集成测 DB 用 aiosqlite in-memory** — M4.6 集成测目标是验证路由/鉴权/审计/限流契约，不验证 SQL 兼容（那是 alembic migration 测的事）。同时 conftest 注入 `FakeRedis` stub 避免依赖真实 Redis。
7. **rate_limit Depends 优先读 `request.app.state.redis`** — 让集成测能直接注入 stub，prod 时 fallback `get_redis()` 单例。
